# Portfolio v2

## Description

**Portfolio v2** est un projet personnel conçu pour présenter mes compétences et mes réalisations en tant que développeur web. Ce portfolio est construit avec des technologies modernes pour offrir une expérience utilisateur fluide et réactive.

## Technologies utilisées

- **Astro**: Un framework pour construire des sites web rapides et optimisés.
- **React**: Une bibliothèque JavaScript pour créer des interfaces utilisateur dynamiques.
- **Tailwind CSS**: Un framework CSS utilitaire pour un design rapide et personnalisé.
- **TypeScript**: Un sur-ensemble de JavaScript qui ajoute des types statiques pour une meilleure robustesse du code.
- **MagicUI**: Une bibliothèque de composants UI pour des éléments interactifs et esthétiques.

## Installation

Pour cloner et exécuter ce projet, suivez ces étapes :

1. **Clonez le dépôt** :
   ```bash
   git clone https://github.com/zdossantos/portfolio-v2.git
   ```

2. **Accédez au répertoire du projet** :
   ```bash
   cd portfolio-v2
   ```

3. **Installez les dépendances** :
   ```bash
   pnpm install
   ```

4. **Démarrez le serveur de développement** :
   ```bash
   pnpm dev
   ```

5. **Ouvrez votre navigateur** et accédez à `http://localhost:4321` pour voir votre portfolio en action.
